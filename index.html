Menyelidiki potensi cacat dan mendiskusikannya dengan pengembang dan melaporkan hasil test ke manager proyek.


Sangat antusias dalam pembuatan User Interface untuk Website atau Mobile Apps, berpengalaman dalam menggunakan software UI design seperti Figma dan Adobe XD, Prototyping dan membuat Wireframe.
